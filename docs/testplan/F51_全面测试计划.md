# F5.1 (stockpush) 全面测试计划

## 测试环境

- **数据库**: PostgreSQL (PGConnector)
- **自选股标的**: 从 `tb_stock_pool` 动态获取
- **历史数据范围**: 最近 1 个月
- **当日测试**: 当前日期 `$(date +%Y-%m-%d)`
- **数据源**: 配置的 primary 数据源 (config.local.yaml → datasources.primary)

## 运行方式

```bash
cd /opt/stockpush

# 直接运行单个测试
python -m pytest docs/testplan/tests/test_01_system.py -v

# 运行全部测试
python -m pytest docs/testplan/tests/ -v

# Hermes API 状态查询
python -m stockpush.worker --hermes status

# 交互式控制台
python -m stockpush.worker
```
