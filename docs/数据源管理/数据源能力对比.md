# 数据源能力矩阵（stockpush F5.1 版）

> 适用于 stockpush 实时信号系统的数据源能力对比。  
> primary 数据源通过 `config.local.yaml` → `datasources.primary` 配置。

---

## 股票数据能力

| 数据源 | 历史日线 | 历史分钟 | 实时日线 | 实时分钟 | 股票列表 | 交易日历 |
|--------|:--------:|:--------:|:--------:|:--------:|:--------:|:--------:|
| XTick | ✅ | ✅ | ✅ | ✅ | ✅ | ✅ |
| Sina | ✅ | ❌ | ✅ | ❌ | ✅ | ❌ |

## 基金数据能力

| 数据源 | 历史日线 | 历史分钟 | 实时日线 | 实时分钟 | 基金列表 | 交易日历 |
|--------|:--------:|:--------:|:--------:|:--------:|:--------:|:--------:|
| XTick | ✅ | ✅ | ✅ | ✅ | ✅ | ✅ |

---

## stockpush 使用的数据源

| 数据源 | 用途 | 说明 |
|--------|------|------|
| **XTick (primary)** | 实时行情 + 历史数据 | 默认 primary 数据源，支持全品类 |
| PostgreSQL | 持久化存储 | `tb_raw_5m/30m/1d` 表存储拉取的行情数据 |

> XTick 为当前 primary 配置，位于 `/opt/stockpush/stockpush/src_mgr/xtick_provider.py`。

## 标准 API 映射

| API 方法 | 说明 |
|----------|------|
| `fetch_stock_history_daily()` | 股票历史日线 |
| `fetch_stock_history_minute()` | 股票历史分钟线 |
| `fetch_stock_realtime_daily()` | 股票实时行情 |
| `fetch_stock_realtime_minute()` | 股票实时分钟线 |
| `fetch_fund_history_daily()` | 基金历史日线 |
| `fetch_fund_realtime_daily()` | 基金实时行情 |
| `fetch_stock_list()` | 股票代码列表 |
| `fetch_trade_calendar()` | 交易日历 |
| `fetch_latest_dividend_date()` | 最新分红日期 |

> stockpush 主要通过 `DataFetcher` → `SrcMgrService` → `StandardAPIAdapter` 调用这些 API。
